# 2.getPrime еҮҪж•°

getPrime еҮҪж•°з”ЁдәҺз”ҹжҲҗ n дҪҚзҡ„йҡҸжңәзҙ ж•°пјҢе…¶дёӯ n дҪҚжҢҮзҡ„жҳҜиҝҷдёӘж•°дәҢиҝӣеҲ¶дёӯеҢ…еҗ« n дёӘдәҢиҝӣеҲ¶дҪҚпјҢеңЁдҝқеӯҳж—¶д»ҘеҚҒиҝӣеҲ¶ (й•ҝж•ҙеҪў) еӯҳеӮЁгҖӮ

# 3.powmod еҮҪж•°

powmod (m,e,n) и®Ўз®— m зҡ„ e ж¬Ўе№Ӯе№¶еҜ№ n еҸ–жЁЎпјҢиҝҷжҳҜ RSA еҠ еҜҶж ёеҝғи®Ўз®—пјҢеёёи§ҒдәҺ RSA еҠ еҜҶдёӯгҖӮ

# 4.RSA

RSA жЁЎж•° nпјҢеҜҶж–Ү cпјҢе…¬й’ҘжҢҮж•° eпјҢ

з§Ғй’Ҙ d дёә П•(n)=(pвҲ’1)(qвҲ’1)пјҢиҝҷдёҖжӯҘйҖҡеёёйңҖиҰҒдҪҝз”Ё yafu,yafu-x64 factor (n) еҲҶи§Ј

d=eвҲ’1 ж¬Ўж–№ modП•(n) d=inverse (e)

еҫ—еҲ°з§Ғй’ҘеҗҺпјҢеҸҜд»Ҙд»Ҙе“ҰйӮЈдёӘжқҘи§ЈеҜҶеҜҶж–Ү c:

m=c зҡ„ d ж¬Ўж–№ mod n

# gmpy2:

жҳҜдёҖдёӘй«ҳзІҫеәҰж•°еӯҰеә“

gmpy2.iroot(n,2): и®Ўз®— n зҡ„е№іж–№ж №е№¶иҝ”еӣһдёҖдёӘе…ғз»„ (root (и®Ўз®—еҗҺзҡ„еҖј),is_exact (еёғе°”еҖј))

gmpy2.gcdext: иҝ”еӣһдёҖдёӘе…ғз»„пјҢз¬¬дёҖдёӘеҖјдёәжңҖеӨ§е…¬еӣ ж•°пјҢз¬¬дәҢдёӘеҖјдёәиҙқзҘ–зӯүејҸзҡ„зі»ж•° x, з¬¬дёүдёӘеҖјдёәиҙқзҘ–зӯүејҸзҡ„зі»ж•° yгҖӮ

# е…ұдә«зҙ ж•°:

RSA еҠ еҜҶж—¶иҝӣиЎҢдәҶдёӨж¬ЎеҠ еҜҶпјҢе№¶дё”з»ҷеҮәдәҶеҠ еҜҶй’Ҙ e, дёӨж¬ЎеҠ еҜҶзҡ„ n1 е’Ң n2, еҜҶж–Ү c, еҲҶеҲ«дҪҝз”Ё n1 е’Ң n2пјҢиҝҷдёӘж—¶еҖҷе°ұеҸҜд»ҘзңӢзңӢдёӨдёӘ n д№Ӣй—ҙжҳҜеҗҰеӯҳеңЁе…ұз”Ёзҡ„зҙ ж•°гҖӮq=gmpy2.gcd (n1,n2)

# е…ұжЁЎж”»еҮ»пјҡ

еҪ“ e1пјҢe2 дә’иҙЁж—¶пјҢgcd (e1,e2)=1, йӮЈд№ҲдёҖе®ҡеӯҳеңЁж•ҙж•° x,y дҪҝеҫ— gcd (a,b)=ax+by,

жүҖд»Ҙеҫ—еҲ° e1 * s1+e2 * s2=1

еӣ дёә e1 е’Ң e2 дёәжӯЈж•ҙж•°пјҢжүҖд»Ҙ s1гҖҒs2 зҡҶдёәж•ҙж•°пјҢе…¶дёӯдёҖдёӘдёәжӯЈдёҖдёӘдёәиҙҹ

жҺЁеҜјиҝҮзЁӢ:

иҝҷйҮҢйңҖиҰҒз”ЁеҲ°дёӨжқЎе№Ӯиҝҗз®—зҡ„жҖ§иҙЁпјҡ

(a * b) % p = (a % p * b % p) % p
a ^ b % p = ((a % p) ^ b) % p

еӣ дёәc1 = m^e1%nпјҢc2 = m^e2%nпјҢйңҖиҰҒиҜҒжҳҺm=(c1^s1*c2^s2)%n

д»Је…ҘеҸҜеҫ—пјҡ

(c1^s1*c2^s2)%n = ((m^e1%n)^s1(m^e2%n)^s2)%n

                             =((m^e1%n)^s1*(m^e2%n)^s2)%n

                             =((m^e1)^s1%n*(m^e2)^s2%n)%n   //ж¶ҲжҺү%n

                             =((m^e1)^s1*(m^e2)^s2)%n

                             =((m^(e1*s1)*(m^(e2*s2))%n   //е№Ӯзҡ„д№ҳж–№пјҢеә•ж•°дёҚеҸҳпјҢжҢҮж•°зӣёд№ҳ

                             =(m^(e1*s1+e2*s2))%n   //еҗҢеә•ж•°е№Ӯзӣёд№ҳпјҢеә•ж•°дёҚеҸҳпјҢжҢҮж•°зӣёеҠ 

еҸҲеӣ дёәm<nпјҢжүҖд»Ҙ(c1^s1*c2^s2)%n=m%n=m

жүҖд»Ҙ m дёә c1 зҡ„ t ж¬Ўж–№ * c2 зҡ„ z ж¬Ўж–№ mod n дёҚйңҖиҰҒеҺ»жұӮ phi

д№ӢжүҖд»ҘеҸҜд»Ҙиҝҷж ·и®Ўз®—пјҢжҳҜеӣ дёәеҪ“еңЁ mod n зҡ„жғ…еҶөдёӢпјҢе’Ң c1 дә’дёәйҖҶе…ғзҡ„ж•°еҸҜд»ҘеҪ“дҪңжҳҜ c1 зҡ„еҖ’ж•°пјҢиҖҢдёҖдёӘж•°зҡ„иҙҹжҢҮж•°ж¬Ўж–№зӯүдәҺдёҖдёӘж•°жҢҮж•°ж¬Ўж–№еҲҶд№ӢдёҖгҖӮ
