# BurpSuite

若要使用 BurpSuite 发送 POST 请求

则改 GET /login HTTP/1.1 为 POST /login HTTP/1.1

并加上 Content-Type: application/x-www-form-urlencoded

若是发送 JSON 格式，则改为 Content-Type: application/json

{

"username": "your_username",

"password": "your_password"

}

注意键值双引号

Content-Type 描述
application/x-www-form-urlencoded 和 GET 查询字段的写法一样，开头不需要 ? ，用 & 符号连接各查询参数，遇到特殊字符需要进行转义。
application/json Body 给出一个 JSON 格式的数据，服务端会解析它。
multipart/form-data 表单字段，一般用于有文件等复杂类型的场景。
